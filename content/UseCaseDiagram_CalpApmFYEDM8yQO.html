  Usuario no encontrado : Use Case

- El administrador intenta modificar/eliminar un usuario que no existe

 

- El sistema muestra mensaje "Usuario no encontrado o eliminado"

 

- El sistema actualiza la lista de usuarios automáticamente

 

- El administrador puede buscar otro usuario o continuar con la gestión

     

 

  Permisos Insuficientes para Gestionar Roles : Use Case

- El sistema detecta que el administrador no tiene permisos para gestionar roles

 

-El sistema muestra mensaje "No tiene permisos suficientes para gestionar roles del sistema"

 

- El sistema redirige al administrador a su panel principal

     

 

  Gestión de Pedidos Problemáticos : Use Case

- El sistema detecta y marca pedidos con problemas:

 

  • Pagos pendientes de confirmación
  • Productos sin stock suficiente
  • Direcciones de envío inválidas
  • Pedidos pausados por tiempo prolongado

  Correo ya existe : Use Case

- El sistema detecta que el email ingresado ya existe en la base de datos

 

- El sistema muestra mensaje "Este email ya está registrado"

  Consulta Sin Resultados : Use Case

- El sistema no encuentra usuarios que coincidan con los criterios

 

- El sistema muestra el mensaje "No se encontraron usuarios con los criterios especificados"

 

- El sistema sugiere ampliar los criterios de búsqueda

 

- El proveedor puede modificar los parámetros
